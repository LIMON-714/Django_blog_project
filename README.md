# Django_blog_project
this is simple django blog project to using python 
Requirements Python 3.x Django 3.x Installation Clone this repository to your local machine using git clone . Navigate to the project directory using the cd command. Install the required packages by running pip install -r requirements.txt. Run the Django development server by running python manage.py runserver. Open a web browser and navigate to http://localhost:8000/ . Usage In order to create, edit, and delete posts, you will need to log in with a registered account.
